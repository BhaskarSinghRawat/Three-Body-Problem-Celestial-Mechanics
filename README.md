# Three-Body-Problem-Celestial-Mechanics

This repository contains all my work done on three body problem in celestial mechanics as an Research intern under the guidance of Dr Md. Chand Asique, Department of Physics, Deshbandhu College, University of Delhi 
In three body problem, we understand the motion of the system, which includes three bodies under the influence of the gravitational forces of each other.

1. Code files
2. Presentation
3. Thesis file
